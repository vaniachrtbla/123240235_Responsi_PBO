# Sistem Manajemen Perpustakaan

Project Responsi Praktikum Pemrograman Berorientasi Objek (PBO)  
Menggunakan Java Swing, MySQL, dan arsitektur MVC.

---

## Deskripsi Project

Project ini merupakan aplikasi desktop Sistem Manajemen Perpustakaan yang dibuat untuk membantu pengelolaan data buku dan majalah secara lebih efisien.

Aplikasi mendukung fitur:
- Login dan Register
- CRUD Buku dan Majalah
- Peminjaman dan Pengembalian Buku
- Histori Peminjaman
- Pencarian Data
- Validasi Input
- Custom Exception
- Koneksi Database MySQL

Project dibangun menggunakan konsep:
- Object Oriented Programming (OOP)
- MVC (Model View Controller)
- Java Swing GUI
- JDBC MySQL Connector

---

## Teknologi yang Digunakan

- Java
- Java Swing
- MySQL
- JDBC Connector
- NetBeans IDE
